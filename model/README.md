# Model

The original modelling (data merge, genre bucketing, class-split
experiments, feature engineering, model comparison) lives in the following notebook in this folder: `MSE_446_project.ipynb`. This was created by author Faisal Alnajar while taking MSE 446 at the University of Waterloo in Fall 2025.

Some further data engineering was done in an attempt to improve model accuracy and better its needs to this project. This is seen in data/prepare_data.py where the outputted csv from the raw files from Kaggle are merged in a similar manner as done in the notebook.
